# coc-please

Please language server extension using plz tool lps for coc.nvim

## Install

`:CocInstall coc-please`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-please-keymap)`

## Lists

`:CocList demo_list`
